Simply convert english numbers into bangla numbers.

== Description ==

This plugin display bangla date and time in post comments.

== Installation ==

1. Download the Plugin 'bangla-date-and-time.zip'

2. Extract and upload the plugin in your 'wp-content/plugins/' dirrectory.

3. Activate the plugin through the Plugins menu.

4. Open any of your post and see bangla date and time in comment section.
