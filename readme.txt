Simply convert english numbers into bangla numbers and english months into bangla.


== Description ==

This plugin display bangla date in comments area and published post.

Visit [author's site](http://www.mhmithu.com/) for exapmle.


== Installation ==

1. Download the Plugin `bangla-date-and-time.zip`

2. Extract zip and upload the plugin in `wp-content/plugins/` directory.

3. Activate the plugin through the plugins menu.

4. Open any of your post and see bangla date and time in comment section.
