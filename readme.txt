Bangla Date and Time simply converts english number into bangla number including twelve english months name.


== Description ==

This plugin simply display bangla date and time in comments area and all other date or time into bangla.

See [author's homepage](http://www.mhmithu.com/) for example.


== Installation ==

1. Download the Plugin `bangla-date-and-time.zip`

2. Extract zip and upload the plugin in `wp-content/plugins/` directory.

3. Activate the plugin through the plugins menu.

4. Open any of your post and see bangla date and time in comments area.
